# Docs Folder

Use this folder for project context that helps AI and humans make better decisions.

## What to put here
- Domain concepts and glossary
- System architecture notes
- Data model and API overviews
- Operational runbooks and constraints

## Suggested approach
- Prefer concise, high-signal docs
- Keep docs current with code changes
- Link to source-of-truth specs when available
- Separate enduring context from temporary plans
