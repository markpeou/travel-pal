# Prompts Folder

Use this folder for reusable, task-oriented prompt templates.

## What to put here
- Feature implementation templates
- Refactor and cleanup workflows
- Bug triage and debugging checklists
- Release or PR preparation prompts

## Suggested approach
- Keep prompts outcome-focused
- Define inputs, constraints, and expected output
- Favor small, composable templates
- Version prompts as team practices evolve
